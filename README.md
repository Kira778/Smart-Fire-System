# 🔥 Smart Fire Detection System

## 📌 Overview

Smart Fire Detection System is an IoT-based safety system designed to detect fire and gas leakage using sensors connected to Arduino.
When danger is detected, the system triggers an alarm and can send alerts (depending on configuration).

---

## 🧠 System Architecture

The system consists of:

* Arduino microcontroller
* Gas Sensor (MQ-2)
* Flame Sensor
* Buzzer
* (Optional) WiFi Module for notifications

---

## ⚙️ Features

* 🔥 Detects flame
* 💨 Detects gas leakage
* 🚨 Triggers buzzer alarm
* 📡 Can be extended to send real-time alerts

---

## 🛠️ Technologies Used

* Arduino (C/C++)
* Embedded Systems
* Basic IoT Concepts
